What is This?

This is collapsible content with a heading and a inner content with good SEO.

Components of this widget?

  1. Heading
  2. Section
Heading can be of two types <h2> and <h3> for this model.
Under <section> you could use any other html tag like <p>, <img>, etc

How to use this in your HTML Page

See the Code Writing Section

Code Writing?

Anywhere in your html page write the code below this: 
<div class="coll-sec"></div>
  <h2>Write your Heading Here</h2>
  <section>Write your Content</section>
</div>
OR 
<div class="coll-sec"></div>
  <h3>Write your Heading Here</h3>
  <section>Write your Content</section>
</div>
And at the bottom of your html page add this external javascript

<script src="https://spdfedu.github.io/pages/hidden-found.js"></script>
